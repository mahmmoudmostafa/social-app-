# Social App

A modern social networking application built with **React** that allows users to connect, share content, and interact in real time.

## Features

* User authentication (login & registration)
* User profile creation and customization
* Real-time social feed for posts and updates
* Follow and unfollow system
* Direct messaging between users
* Real-time notifications
* Responsive mobile-friendly design

## Tech Stack

* **Frontend:** React 18, React Router, Redux
* **Styling:** CSS3, Tailwind CSS / Bootstrap
* **API:** RESTful APIs
* **Build Tools:** Vite / Create React App
* **Package Manager:** npm / yarn

## Installation

```bash
git clone <repository-url>
cd social-app
npm install
npm start
```

Create a `.env` file:

```
REACT_APP_API_URL=your_api_url
REACT_APP_AUTH_TOKEN=your_token
```

Then run the app on:

```
http://localhost:3000
```
